# FamilBudget v1.0.0

Udgivet den 2026-07-19

## Nyheder
- Opdateret desktop-oplevelse med indbyggede panels i stedet for nye vinduer
- Forbedret struktur til release og distribution

## Fejlrettelser og forbedringer
- Bedre brugeroplevelse på Windows
- Forberedt release-pakke til GitHub Releases

## Installation
1. Download `budget.exe` eller `FamilBudget-v1.0.0.zip`
2. Pak arkivet ud hvis nødvendigt
3. Start `budget.exe`

## GitHub Releases
- Upload `budget.exe` som en release-asset for opdatering via appen
- Upload `FamilBudget-v1.0.0.zip` som den fulde pakke
- Upload `BudgetManagerSetup.exe` hvis du benytter installationsprogrammet
